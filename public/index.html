O QUE ELE CAPTURA

Tudo que a auditoria analisa

40+ verificações automáticas organizadas em categorias de segurança

🔑

Credenciais Expostas

  • Arquivos .env públicos com conteúdo completo
  • Service Keys e tokens no bundle JS
  • Anon Keys expostas no código-fonte
  • Segredos em comentários HTML
  • Senhas em variáveis de ambiente visíveis
🛡️

Vulnerabilidades de API

  • REST API sem autenticação
  • RPC functions executáveis sem token
  • GraphQL introspection aberta
  • Edge Functions com acesso livre
  • Endpoints que vazam dados internos
🔒

Controle de Acesso

  • Row Level Security (RLS) desativado
  • Tabelas acessíveis sem autenticação
  • Storage buckets públicos indevidos
  • Dados de clientes expostos (PII)
  • Permissões de função excessivas
🌐

Headers & Configurações

  • Headers de segurança ausentes
  • CORS configurado incorretamente
  • Content-Security-Policy fraco
  • HSTS não implementado
  • Clickjacking sem proteção
⚡

Ataques & Resiliência

  • DDoS e DoS (Slowloris, ReDoS)
  • Brute Force sem rate limiting
  • Simulação de ataque Hydra
  • Port Scan em serviços expostos
  • HTTP Request Smuggling
🔍

Código-Fonte & Rotas

  • Análise profunda de JS/bundles
  • Rotas ocultas e endpoints secretos
  • Git exposure (.git acessível)
  • Open Redirect vulnerável
  • Comentários com dados sensíveis
🔐

SSL/TLS & Certificados

  • Validade e configuração SSL
  • Protocolos obsoletos (TLS 1.0/1.1)
  • Cipher suites inseguros
  • Certificados autoassinados
  • Mixed content HTTP/HTTPS
🕵️

Dados Sensíveis & PII

  • CPF, CNPJ, cartões em APIs abertas
  • E-mails e telefones expostos
  • Endereços e chaves Pix visíveis
  • Dados de clientes SaaS expostos
  • Informações médicas ou financeiras
